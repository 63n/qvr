void QVROutputPlugin ( QVRWindow *  window,

)

Output the rendered content.

Parameters
windowThe window that provides the output
contextThe render context of the output
tex0The output of view pass 0
tex1The output of view pass 1 (if any)

This is the main function of an output plugin. It takes the rendered output from the textures tex0 and tex1 and displays it to the output device, applying arbitrary postprocessing.

The context provides all information available about the output. Note that tex1 is only valid if the output mode of window has two view passes. This can be checked with QVRRenderContex::viewPasses().

See also QVROutputPluginInit() and QVROutputPluginExit().
