const QString& QVRNavigatorConfig::parameters ( ) const
inline

Returns parameters (may be empty).

Specific navigator types may require parameters.

For VRPN navigators, this is currently a string of the form <name> [<sensor>], where <name> is the VRPN tracker name, e.g. Tracker0@localhost, and <sensor> is the sensor number to be used. The sensor number can be omitted.
